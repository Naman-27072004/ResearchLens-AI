# Security Policy

## Reporting a Vulnerability

We take the security of ResearchLens AI seriously. If you find a security vulnerability, please do **NOT** open a public GitHub issue. Instead, report it privately to the maintainers:

*   Email: **security@example.com** (Please replace with your actual contact email when configuring the repo)

Please include the following details in your report:
- A description of the vulnerability and its potential impact.
- Step-by-step instructions or proof-of-concept code to reproduce the issue.
- Details about your environment/configurations.

We will review your report and respond as quickly as possible. Once the issue is resolved, we will publish a security advisory and credit you for the discovery.
